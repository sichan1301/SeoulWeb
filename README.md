## 😎 개요

- 한국교통대학교 컴퓨터공학과 캡스톤디자인 (졸업작품)
- 서울의 지리, 관광 명소, 축제 등을 소개하고 소통하는 사이트
- 로그인, 게시판 기능 (CRUD, Pagination)
- 코로나 확진자 현황, 날씨, 주요 지하철역 실시간 상황 제공

## 😊 기술스택

- HTML / CSS
- Bootstrap
- Python, Django
- Crawling (Python, BeautifulSoup, JSON, XML)
- 공공데이터 APIs
